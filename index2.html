Welcome to my data science portfolio! This collection of projects showcases my analytical skills and my ability to turn complex data into actionable insights. MY PROJECTS

Data Science Projects

Assessing ChatGPT's Impact on Business, Healthcare, and Workforce

This project explores the impact of ChatGPT, a Generative Pretrained Transformer (GPT) developed by OpenAI, on various aspects of society, including business enterprises, healthcare, workforce, and ethical considerations.

Leveraging Data Science to Analyze Student Performance

This project aims to analyze and understand the factors influencing student performance, stress, and academic success using data science methods.

Evaluating Housing Affordability

The objective of this project is to analyze and predict rental prices based on various factors such as area, number of bedrooms, and availability of amenities like power backup.

Unveiling the Social Progress

The primary objective of this project is to analyze the relationships between crime rates, corruption perceptions, and social progress indices across various countries to identify socio-economic patterns and insights that may influence policy-making and development strategies.

Assessing Loan Default Risks

The objective of this project is to develop a comprehensive predictive model that identifies the likelihood of loan defaults based on various borrower characteristics.

Harnessing Predictive Analytics in Real Estate

The objective of this project is to predict the sale prices of residential properties by analyzing various features and characteristics that influence housing prices.

The Childcare Affordability Crisis

This comprehensive data analysis project investigates the critical factors affecting child care affordability across the United States, addressing a pressing challenge that impacts family stability, workforce participation, and economic growth.

Predicting US Traffic Accident Severity

This project aims to addresses the critical challenge faced by auto insurance companies, public safety agencies, and transportation departments in transitioning from reactive accident response to proactive prevention and resource optimization.

Exploring Trends in Movie Performance

The objective of this project aims to build and evaluate machine learning models that predict whether movies will receive high ratings (above 7.0).

Classifying Anomalous Financial Transactions

This project aims to develop a machine learning model that classifies financial transactions as fraudulent or legitimate by analyzing historical transaction data and identifying patterns indicative of fraud.
